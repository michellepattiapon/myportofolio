About

Selamat datang di halaman web saya. Saya berharap kalian menikmati kunjungan kalian di sini dengan mengenal siapa saya, terima kasih telah mengunjungi halaman web saya. Nama lengkap saya Michelle Stephanie Pattiapon, namun biasa disapa Michelle atau Fanny. Saya Lahir di Kota Sorong, Papua tepatnya pada tanggal 06 April 2000. Lahir dan besar disana selama 13 tahun, saya pindah ke Ambon pada tahun 2013 saat saya naik kelas 8. Sedari kecil saya aktif disekolah maupun di lingkungan tempat tinggal saya. Hobby saya adalah menyanyi dan memotret, tidak heran saya sedari kecil saya telah dibiasakan dengan lomba-lomba menyanyi dan bergabung dengan paduan-paduan suara anak dan remaja sedari kecil, hingga berlanjut saat ini dalam paduan suara pemuda maupun pesparawi di tempat saya tinggal.

Kalau hobby memotret baru saya sadari ketika saya berada di bangku SMA. Saya senang mengambil gambar segala sesuatu yang ada disekitar saya, memotret orang maupun pemandangan. Dibawa ini, sudah saya masukkan beberapa dokumentasi kegiatan-kegiatan yang biasa saya ikuti maupun hasil potret yang saya miliki, hehehe. Saat ini, saya berstatus Mahasiswa Program Studi Pendidikan Matematika pada Universitas Pattimura. Alasan saya memilih Pendidikan Matematika karena saya senang mendedikasikan diri saya untuk orang lain, dalam hal ini mengajar adalah jalan yang saya pilih. Dan kenapa Matematika? karena saya senang berhitungan dan merasa tertantang dengan jurusan satu ini. Yah, semoga di esok hari saya dapat mengajak orang lain untuk mencintai Matematika.

Halaman web ini saya buat sebagai Tugas salah satu matakuliah saya yaitu Pemograman web. Sekian dulu perkenalan dengan saya, semoga kalian senang mengenal saya. Bye byeeeee
